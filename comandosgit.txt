git init - Inicializa

git add reademe.md - Adiciona um arquivo no caso "reademe.md"

git status  - Status dos comites e dos arquivos comitados

git commit -m "primeiro commit" - Como realizar o primeiro comite

git branch -M "main" - Trocando de Master para Main

git remote add origin "Link GitHub" - Levar seus arquivos para o github na nuvem
